Sistema de Referência

O que é um sistema de referência?

Na Física é de suma importância adotarmos um certo referencial, pois é a partir dele que conseguiremos definir a posição de um objeto qualquer. Inicialmente aprendemos sistemas de referências retangulares também chamados de cartesianos em homenagem à Renè Descartes, dependendo do problema podemos lançar mão de sistemas que possuem outras geometrias como são os casos dos sistemas cilíndricos e esféricos.

Como podemos criar um sistema de referência?

Para criarmos um sistema de referência devemos designar seu ponto inicial, ou seja, o ponto a partir do qual traçaremos os eixos.

Podemos criar qualquer sistema de referência?

A princípio sim, porém é bom seguirmos o padrão existente para que possamos mudar de um sistema para outro, no caso de sistemas cartesianos devemos seguir os seguintes passos para chegarmos ao par, ou trio, que definem um ponto:

  • Definir o ponto inicial;
  • Traçar as retas que, por padrão, devem ser ortogonais entre si;
  • A sequência das retas ortogonais devem seguir a sequência segundo o sentido Anti Horário.
